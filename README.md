# Reinforcement Learning (Q Learning)

This repository contains two implementation of q-learning in different scenarios:
- Robot in a Room
- OpenAI gym's LunarLander-v2

## Dependencies
### Python
[python]

### Gym
```console
$ pip install gym[all]
```

## Quick Start
### Robot in a Room
```console
$ python robot_q_learning.py
```
### LunarLander-v2
```console
$ python lunarlander_qlearning.py
```
